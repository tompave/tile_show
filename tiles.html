Tile Show! (try and resize the window)

Except for one, I don't own any of the artwork here displayed. All images belong to their respective authors. Links to the original locations of the images and to the authors' webpages are provided within each tile. All authors have been contacted and have granted permission to use their works, with attribution.

Resize the browser window to have the tiles fluidly rearrange. Hover the tiles to reveal additional information. On touchscreens, the hover event is replaced by a single-tap trigger.

This page is intended as a mere demo of a design pattern and its implementation. The source code is freely available on GitHub. For info, head to wonderingmachine.com.
